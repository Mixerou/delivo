# Delivo Web Client

Web Client module of the Delivo Project

## Available Commands

```bash
# Prepare environment
$ pnpm run postinstall

# Start the development server on http://{instance address}:3000/
$ pnpm run dev

# Generate static version
$ pnpm run generate

# Generate production build
$ pnpm run build

# Preview project
$ pnpm run preview

# Check code (pnpm only)
$ pnpm run plint

# Fix code (pnpm only)
$ pnpm run plint:fix
```
